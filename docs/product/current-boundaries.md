# 当前产品边界

这个文件只记录今天仍然有效、仍然约束开发和评审的产品边界。

它不是路线图，也不是历史版本说明。

## 当前产品形态

Rebase 当前是一个内容驱动的社区网站，加上一套供工作人员使用的内部管理工作台。

核心事实：

- 公共网站面向读者，只读
- 内容维护、发布、归档和配置修改通过内部管理工作台完成
- 公共网站不承担读者登录、成员系统或公开提交流程

## 当前公共界面范围

当前应持续维护的公共界面包括：

- 首页
- About 页面
- `Who-Is-Hiring` 列表与详情
- GeekDaily 列表、详情与搜索
- 文章列表与详情
- 活动列表与详情
- 贡献者列表
- 全局页脚
- RSS 输出

## 当前内容与运营边界

- 内容编辑以结构化字段加 Markdown 为主
- GeekDaily 搜索是当前唯一明确在范围内的站内内容搜索能力
- 工作人员可以维护内容、媒体、站点设置、角色权限和审计可见性
- 公共页面只消费已发布内容

## 当前明确不做的事情

以下内容当前仍不在产品范围内：

- 读者账号、成员登录和公开用户系统
- 评论、点赞、收藏、通知
- 站内活动报名
- 站内职位申请
- 多语言内容系统
- 复杂审批流
- 实时协同编辑
- 可视化页面搭建器
- 全站统一搜索

## 使用规则

如果一项变更会扩大、缩小或改写上面的边界，必须在同一个 commit 中同步更新：

- 本文件
- `docs/product/content-model.md`
- 相关的架构或运维文档
